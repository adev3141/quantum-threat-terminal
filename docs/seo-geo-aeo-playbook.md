# XQBTS SEO / GEO / AEO Playbook

This document captures the off-page authority and distribution work needed in addition to technical/on-page changes.

## Priority keyword themes

1. Quantum threat intelligence
2. Q-Day readiness and countdown modeling
3. Post-quantum cryptography migration risk
4. Harvest Now, Decrypt Later (HNDL)
5. Quantum benchmark evidence and curation
6. Quantum compilation website / quantum computing security dashboard

## Off-page authority plan

### 1) High-trust backlinks (monthly)
- Publish methodology explainers on Medium + LinkedIn and link to `/whitepaper`.
- Submit technical commentary to cybersecurity outlets and quantum newsletters.
- Contribute guest posts to PQC / AppSec communities with canonical links to `xqbts.com`.

### 2) Entity building
- Maintain consistent Organization identity: `XQBTS` by `3DDev SMC PVT`.
- Create and maintain profiles (LinkedIn, GitHub, Crunchbase, Product Hunt) with canonical URL `https://xqbts.com`.
- Use the same brand description and keyword set to strengthen entity resolution in search and AI assistants.

### 3) Digital PR cadence
- Ship one “benchmark methodology update” post per month.
- Ship one “state of Q-Day readiness” post per quarter.
- Every post should include structured summaries and FAQ snippets to support answer engines.

## GEO (Generative Engine Optimization) tasks

- Keep `public/llms.txt` updated with key URLs and terminology.
- Keep schema.org JSON-LD current (Organization, WebSite, FAQPage).
- Keep concise, high-signal definitions on the homepage for LLM retrieval.
- Ensure whitepaper sections use explicit headings for chunk-level retrieval.

## AEO (Answer Engine Optimization) tasks

- Expand FAQ entries as new user questions appear in analytics.
- Ensure each FAQ answer is direct, factual, and 1-3 sentences.
- Reuse FAQ blocks in social and docs to reinforce answer consistency.

## Measurement

- Track impressions/clicks for target keyword clusters in Search Console.
- Track rich result eligibility for FAQ + organization markup.
- Monitor AI answer references manually (ChatGPT, Perplexity, Gemini) on top keyword prompts.
- KPI targets: +30% non-brand impressions in 90 days, top-10 rank for at least 5 target phrases.
